

# Rotten Potatoes

Rotten Potatoes is a web application like rotten tomatoes built using ruby on rails framework

**Rotten Potatoes** built with [Ruby][0] using the [Ruby on Rails Web Framework][1].


## Technology Stack

- [Ruby][0]
- [Ruby on Rails][1] 
- [Bootstrap][4] 
- [jQuery][5] 
- [Heroku][6]
- [Travis CI][7]
